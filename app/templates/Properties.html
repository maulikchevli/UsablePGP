  • PublicKey

    Public Key can be known to everyone. It is used to Encrypt the message in the Publickey Cryptography(RSA). We have used 256 bytes length of Public key. Sender needs to encrypt the message by receiver's Publickey in RSA.

  • PrivateKey

    Private Key can not be accesed by other users. it remains confidential for every users. It is used to decrypt the message encrypted by corresponding Publickey of user in Publickey Cryptography(RSA). Private key is of length 256 bytes. Recipient needs to use his own private key to decrypt the encrypted content to view the sender's message.

  • Encryption

    It is used to encode the message or file being sent in the communication so that it can read by only certain people. There are many techniques available for encryption, here we have used RSA Cryptosystem. Public Key is used for encryption.

  • Decryption

    It is the technique to convert the encrypted text to original message/plaintext. Private Key is used for decryption in RSA.

  • Signature

    A digital signature is a technique used to validate the authenticity and integrity of a message or plaintext. Recipient can identify the authenticity of a message received by signature. In RSA, Private key of sender is used to Sign the message. It can be verified by using Public Key of Sender.

  • Key Regeneration

    It is used to generate new Public-Private keypair for the user. Old keypair would be revoked. The step is irreversible. After this, nobody will be able to use your old key for any task.

  • Key Revocation

    It will remove the existing Public-Private keypair of the user as well as the user himself. The step is irreversible. After this, you will not be able to use Usable PGP.

  • Signature Verification

    It is used to verify the sender's identity using RSA signature technique. In RSA, Public Key of user is used for verification.

  • Algorithms Used

    The Algorithms being used in the Usable PGP app for implementation.

    Message Encryption/ Decryption:- RSA

    Signature/ Signature Verification:- RSA
